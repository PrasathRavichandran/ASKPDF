Web application project for loading PDF files and ask/clarify questions on the PDF using integrated AI.

## Getting Started

For running the development server:

```bash
cd Project && bun run dev
```

## Tech Used

- Next JS
- Tailwind CSS
- tRPC 
- Prisma
